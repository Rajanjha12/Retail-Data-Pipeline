Project Description:
This project demonstrates an end-to-end Retail Data Engineering Pipeline built on Azure Databricks. The pipeline ingests raw retail data, performs data cleaning and transformation using PySpark, stores data in Delta Lake, and implements the Medallion Architecture (Bronze, Silver, Gold). The final Gold layer is used to build interactive dashboards in Databricks SQL/Lakeview.

Project Features:
- Data ingestion from CSV files stored in Unity Catalog Volumes
- Data cleaning and type casting
- Bronze, Silver, and Gold data layers
- Delta Lake tables
- ETL pipeline using PySpark
- SQL analytics
- Interactive Databricks Dashboard
