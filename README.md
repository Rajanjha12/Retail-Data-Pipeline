Project Description:
This project demonstrates an end-to-end Retail Data Engineering Pipeline built on Azure Databricks. The pipeline ingests raw retail data from Unity Catalog Volumes, performs data cleaning and transformations using PySpark, stores processed data in Delta Lake, follows the Medallion Architecture (Bronze, Silver, Gold), and provides business insights through Databricks SQL and Lakeview Dashboards.
